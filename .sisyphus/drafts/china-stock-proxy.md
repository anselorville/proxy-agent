# 草稿: A股市场金融数据代理服务

## 需求总结（用户初始描述）

### 核心问题
1. 需要获取中国A股市场金融数据
2. 现有接口（如akshare）存在高频调用IP封禁问题
3. 需要稳定、可靠的数据源

### 核心功能要求
1. **定时/手动数据保存**
   - 定时任务：15:05（A股15:00收盘后）
   - 手动触发：随时可执行
   - 获取主要指数成分股的前复权日线交易数据
   - 包含完整字段（开盘、收盘、最高、最低、成交量、成交额等）
   - 数据回溯：最近3年

2. **数据下载接口**
   - 提供Bearer token认证的API
   - 支持时间区间筛选
   - 支持股票代码范围筛选
   - 部署在阿里云服务器

### 量化分析用途
- 用于量化交易分析的选股策略
- 需要确认复权类型：前复权 vs 后复权

### 技术偏好
- Python + FastAPI + PostgreSQL
- 初始阶段：免费方案试运行
